clocr — コーデ色バランス診断アプリ
⚠️ 現在開発中のデモ版です。

概要
70:25:5カラールール（ベース70%・アソート25%・アクセント5%）に基づき、コーディネート写真をAIで解析してカラーバランスをスコア化するWebアプリです。
手持ちの服をデータベースに登録し、診断結果に合わせた具体的なスタイリングアドバイスも提供します。
授業の個人制作として開発。設計・実装・デプロイすべてを担当。

技術的なポイント
1. MediaPipeによる人物セグメンテーション
selfie_segmenter.tflite を使って人物領域を自動抽出
バウンディングボックスではなく実際の人物ピクセル数で割合を正規化することで、全身・上半身など撮影距離の違いに対応
2. k-meansによる色抽出
OpenCVのk-means（K=6）でHSV空間の色クラスタリング
HSV範囲指定で肌色を除去し、肌色がベースカラーとして誤検出されるのを防止
抽出色を割合順に並べ、1位・2位・3位をベース・アソート・アクセントとして評価
3. 70:25:5スコア計算
diff = |p1-70| + |p2-25| + |p3-5| の合計誤差からスコアを算出
配色タイプ（類似色・補色・モノトーンなど）を色相角度差で判定し、ボーナス点として加算
4. スタイリングアドバイス
スコアの低い要因（ベース不足・差し色過多など）を自動判定してテキスト生成
無彩色かどうかを判定し、「グレーなのにグレーに変えろ」という矛盾したアドバイスを防ぐ
Supabaseに登録した手持ち服の中からベース・アソート・アクセント別に1件ずつ具体的なアイテムを提案
5. Supabaseバックエンド
outfit_logテーブルにコーデ履歴を保存（スコア・色データ・画像URL）
clothesテーブルに服を登録し、診断時のアドバイスに活用
wardrobe-imagesバケットに写真・グラフ画像を保存
使用技術
分類	技術
フロントエンド	Streamlit
人物抽出	MediaPipe（selfie_segmenter）
画像処理・色解析	OpenCV、NumPy
データ可視化	Matplotlib（ドーナツ型パイチャート）
バックエンド	Supabase（PostgreSQL + Storage）
デプロイ	Streamlit Cloud
技術選定理由
技術	選定理由
MediaPipe	Googleが提供する軽量セグメンテーションモデルで追加費用なし。Streamlit Cloud上でも動作する
OpenCV k-means	画像の代表色抽出に実績のある手法。HSV空間での処理により色相・彩度・明度を直感的に操作できる
Supabase	無料枠で画像ストレージ・DBを一括管理できる。RLSの設定で公開範囲のコントロールも容易
Streamlit	PythonのみでUIを構築でき、Streamlit Cloudへの無料デプロイが容易
システム構成
写真アップロード
    ↓
MediaPipeで人物領域を抽出・肌色を除去
    ↓
k-meansで代表色を抽出（K=6）
    ↓
70:25:5スコア計算 ＋ 配色タイプ判定
    ↓
アドバイス生成（手持ち服データと照合）
    ↓
結果表示・Supabaseに保存
機能一覧
ページ	機能
診断する	写真アップロード → スコア・配色タイプ・カラー分析・アドバイス・保存
コーデ履歴	過去の診断結果を一覧表示・編集・削除
服を登録	服の写真から色を自動抽出して服テーブルに登録
服一覧	登録済みアイテムのグリッド表示・フィルタ・編集・削除
うまくいかなかった点
服単体登録の精度：MediaPipeは人物全体を想定したモデルのため、服だけの写真では検出が不安定
同系色の誤分割：グレー44%とグレー41%のように同じ色が別クラスタに分かれ、2色拮抗と誤判定されるケースがある
アドバイスの限界：条件分岐で文章を生成するため、コーデの文脈に合わない指摘になる場合がある
今後改善したい点
服単体の色登録を背景除去専用モデル（rembg など）に変更
手持ち服の提案を色相相性スコアで選ぶよう改善
季節・気温パラメータでフィルタリング（夏は半袖のみ提案など）
コーデの組み合わせ提案（「このトップス×このボトムスでスコアXX点」）
同系色クラスタを統合してより正確な色比率を算出
ローカル実行
1. リポジトリをクローン

git clone https://github.com/yurayura3812-svg/clocr.git
cd clocr
pip install -r requirements.txt
2. .streamlit/secrets.toml を作成

SUPABASE_URL = "your_supabase_url"
SUPABASE_KEY = "your_supabase_anon_key"
3. 起動

streamlit run app.py
